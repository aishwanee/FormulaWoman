**Abstract — Formula Woman Website**

The *Formula Woman* website is an interactive racing platform inspired by modern motorsport websites, built using HTML, CSS, JavaScript, and React JS. It is designed to simulate a complete digital ecosystem for a racing championship, combining clean design with dynamic, real-time features.

**User Authentication (Sign In & Subscribe):**
The platform includes dedicated Sign In and Subscribe features implemented through custom forms. These forms simulate real-world user authentication flows, allowing users to input their details and interact with the platform in a more personalized way. This adds a layer of realism and introduces basic front-end form handling and validation concepts.

**Merchandise and Tickets:**
The website features separate sections for Merch and Tickets, designed as storefront-style pages. Users are redirected from the main interface to these pages, where they can explore products or purchase race tickets. This creates a more complete experience by mimicking the commercial side of real racing platforms.

**Calendar and Schedule System:**
The calendar is one of the core dynamic components of the website. It displays the full racing schedule while also intelligently identifying the previous, next, and upcoming races based on the current date. A live countdown timer is integrated for the next race, updating in real time. Additionally, the platform displays both the track’s local time and the user’s local time for each race, ensuring clarity for a global audience. A dropdown version of the calendar is also included in the navigation bar for quick access.

**Results and Archive:**
The Results section provides race outcomes and standings, along with an Archive feature that allows users to explore data from previous seasons. This adds depth to the platform and enables users to engage with historical performance data, similar to official motorsport websites.

**Drivers and Hall of Fame:**
The Drivers section includes a dropdown menu for quick navigation to individual driver profiles. Each profile presents key details in a visually engaging format. Additionally, a Hall of Fame feature highlights all world championship-winning drivers, showcasing the legacy and achievements within the sport.

**Teams:**
The Teams section is structured similarly to the Drivers section, with a dropdown menu that provides direct access to individual team pages. Each team page represents branding and identity, contributing to the overall realism of the platform.

Overall, the *Formula Woman* website demonstrates how modern web technologies and component-based design using React can be combined to create a scalable, dynamic, and immersive sports platform.
